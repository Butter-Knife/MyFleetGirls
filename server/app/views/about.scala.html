なにこれ

自分の艦これデータをこのサーバに送って公開できるツールです。

艦これやってて「艦これの進捗、他人にサッと見せれたら便利だなー」と思って作りました。

一応他の艦これツールにありがちな遠征時間などのチェック機能も付いてます。

インストール・使い方

  1. JavaVM（JRE）がインストールされてない場合はする
  2. Proxy Server Download
  3. ZIPファイルを解凍
  4. application.confをテキストエディタで開き、自分の艦これサーバのIPアドレスを設定。初期設定はブイン基地になってます
  5. ブラウザのProxy設定
    1. 特定アドレスへの通信にProxyを設定するaddon等をインストール。ChromeならAuto SwichySharp、FirefoxならFoxyProxyで動作確認済
    2. 前に設定した艦これサーバのIPアドレスのアクセスをlocalhost:8080へ転送する
  6. MyFleetGirls.shかMyFleetGirls.batかMyFleetGirls.jarを実行する
  7. 先程設定したブラウザで艦これのFlashを起動する
  8. http://myfleet.ponkotuy.com/u/{ユーザ名} にアクセスすると閲覧できる

注意事項

  • 艦これのデータはぽんこつさんや他の人に丸見えになります。（パスワード類は除く）
  • Token等認証系の情報はこちらの鯖に送らないように気をつけましたが、技術的には全然できてしまうので、ぽんこつさんを信用するか、コード読める人じゃない限りやめましょう
